# Screenshots & Demo

Add your app screenshots and GIFs here!

## How to Add
1. Take a screenshot: `Win+Shift+S` or use Snip & Sketch
2. Save image to this folder (e.g., `screenshot.png` or `demo.gif`)
3. Update README.md to reference the image

Example in README:
```markdown
![SentimentSleuth Demo](docs/screenshot.png)
```
